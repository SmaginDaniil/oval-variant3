# Овал — Вариант 3


Проект реализует требование задания: классы Point и Oval, чтение данных из файла, валидация, вычисления площади и периметра (приближённый) овала, проверка — круг/овал, проверка пересечения только одной оси на заданное расстояние.


## Технологии
- TypeScript
- Node.js (ES Modules)
- pino для логирования
- Jest для тестирования
- ESLint (Airbnb-ish)
